# GIS-Mapping Project

- Course Project in C++ using EZGL and GTK for UI. 
- Features:
    - Autocomplete Search Function
    - Path Finding Between 2 intersections
    - Current Weather
    - Shows your locations based on ip address
    - Shows Public Tranportations Lines
